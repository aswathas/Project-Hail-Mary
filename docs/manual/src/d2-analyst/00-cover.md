# About this document

This handbook is for the analyst running an investigation. It assumes
no Python or Elasticsearch background. It walks through the UI, the
workflow, how to read what ChainSentinel reports, and five complete
case studies — one per shipped simulation.

If you are extending the tool with new detection rules, see **D5
Developer Guide**. If you are responsible for deploying ChainSentinel,
see **D6 Operations Guide**.
