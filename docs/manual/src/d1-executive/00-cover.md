# ChainSentinel in one paragraph

ChainSentinel is a blockchain forensics tool. A client says *"we got
exploited"*, hands over an RPC endpoint, a handful of contract ABIs,
and an approximate time window. ChainSentinel runs a full forensic
pipeline against the chain, looking for the fingerprints of every
attack class it has been taught to recognise — 60 detection signals,
38 attack patterns, fund-trace analysis up to five hops, and a
Gemma-3 powered copilot that writes the report. No cloud dependency.
Works offline. Switches between Anvil simulations, testnets, and
mainnet with a single URL change.
