# About this document

The operations guide for ChainSentinel. Covers installing the stack,
running it via Docker or `start.sh`, every key in `config.json`, Kibana
setup, monitoring, troubleshooting, and upgrade procedures.

Audience: DevOps engineers and analysts responsible for running the
tool on their own infrastructure.
