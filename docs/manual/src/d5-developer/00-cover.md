# About this document

The developer reference for ChainSentinel. Module-by-module API contracts,
extension cookbooks (add a signal, add a pattern, add a derived event,
add an ABI), and the test strategy.

This document assumes you have read **D3 Architecture & Design**. The
APIs documented here implement the architecture described there.

Companion documents:

- **D3 Architecture & Design** — the model these modules implement.
- **D4 Detection Reference** — exhaustive catalogs of everything this
  guide describes the *interface* of.
- **D6 Operations Guide** — how to install, run, and deploy.
- **D7 Simulation Guide** — scenarios that exercise the system end-to-end.
